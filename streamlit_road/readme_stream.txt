First you need to install Streamlit:

pip install Streamlit

Check that Streamlit is installed by typing the command Streamlit hello in the Terminal and by clicking on Enter.
Streamlit hello

A web page should open automatically, with Streamlit demonstrations. You can explore the web page. You can then close it using the ctrl+c command in the Terminal.

go to the folder containing the .py Python script and display all the items in that folder. The created Python script should be displayed.

Create the web application by typing the command streamlit run script.py (where script.py is the name of your Python script) in the Terminal and by pressing Enter.

 streamlit run streamlit_car_3.py 

A web page should automatically open and should contain your interactive Streamlit.
